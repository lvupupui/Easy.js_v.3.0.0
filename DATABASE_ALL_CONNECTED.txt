HOW TO USE - EXAMPLES
════════════════════════════════════════════════════════════════════════════════

Single Database in DSL:

  USE MONGODB mongodb://cluster.mongodb.net/mydb
  
  OR

  USE POSTGRESQL postgresql://user:pass@host:5432/mydb
  
  OR

  USE DYNAMODB aws://region/table-name
  
  OR

  USE FIREBASE projects/my-project/databases/my-db

Multiple Databases in DSL:

  USE MONGODB mongodb://cluster.mongodb.net/primary
  USE ELASTICSEARCH http://elastic:9200
  USE FIREBASE projects/my-project
  
  # Primary: MongoDB
  # Secondary: Elasticsearch for search
  # Tertiary: Firebase for real-time

In JavaScript Code:

  // Query primary database (MongoDB)
  await db.query('users', 'findMany');
  
  // Query specific database (PostgreSQL)
  await db.queryDatabase('postgresql', 'users', 'findMany');
  
  // Switch primary database
  await db.switchPrimaryDB('postgresql');
  
  // Replicate to all databases
  await db.replicateToAll('users', 'create', userData);
  
  // Get list of connected databases
  const dbs = db.getConnectedDatabases();
  // ['mongodb', 'elasticsearch', 'firebase']
  
  // Health check
  const health = await db.healthCheck();

MIGRATION BETWEEN DATABASES
════════════════════════════════════════════════════════════════════════════════

Super easy with unified API:

  // Get all data from MongoDB
  const users = await db.queryDatabase('mongodb', 'users', 'findMany');
  
  // Write to PostgreSQL
  const pgAdapter = db.getAdapter('postgresql');
  for (const user of users) {
    await pgAdapter.query('users', 'create', user);
  }

MULTI-DATABASE SETUP EXAMPLE
════════════════════════════════════════════════════════════════════════════════

Perfect for complex apps:

  // Store user data in PostgreSQL (reliable)
  // Search in Elasticsearch (fast full-text)
  // Real-time in Firebase (live sync)
  
  USE POSTGRESQL postgresql://host/mydb
  USE ELASTICSEARCH http://elastic:9200
  USE FIREBASE projects/my-project
  
  // Create operation
  const user = await db.query('users', 'create', userData);  // PostgreSQL
  
  // Index for search
  await db.queryDatabase('elasticsearch', 'users', 'create', user);
  
  // Broadcast real-time
  await db.queryDatabase('firebase', 'users', 'create', user);
