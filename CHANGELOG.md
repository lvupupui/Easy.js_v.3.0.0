# Changelog

All notable changes to easy.js will be documented here.

## 3.0.0 - 2026-05-01

- Added easy.js language layer: grammar, spec, type checker, formatter, linter, REPL, playground, debugger helpers, source map helper, compiler plugin API, standard library metadata, package registry, and conformance test.
- Added production backend framework features: auth hardening, API keys, webhooks, observability, backups, query builder, compliance helpers, and admin/audit APIs.
- Added IDE support for `.easy`, HTML, CSS, Tailwind, Bootstrap, Material UI, React, Vue, and Svelte.
- Added secure project starter, `doctor`, `add`, `lint`, `format`, `typecheck`, `repl`, and `playground` CLI commands.
- Added official logo and author branding for Avi Ranjan Prasad.
