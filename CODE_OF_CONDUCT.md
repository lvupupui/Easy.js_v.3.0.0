# Code of Conduct

easy.js is built for a welcoming developer community.

## Our Standards

- Be respectful and constructive.
- Help beginners feel capable.
- Assume good intent.
- Keep feedback focused on code, design, documentation, and user experience.
- Do not harass, threaten, insult, or discriminate.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.

## Reporting

Report concerns through the project issue tracker or the maintainer contact listed in the repository.
